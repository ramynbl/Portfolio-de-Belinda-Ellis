# Portfolio de Belinda Ellis

Ce projet est un **site portfolio** réalisé avec **Next.js**. Il présente le travail de **Belinda Ellis**, créatrice d'univers visuels, avec une esthétique minimaliste, des typographies General Sans et Inter Tight, et des animations fluides.

## Démarrage

Installez les dépendances et lancez le serveur de développement :

```bash
npm install
npm run dev
# ou yarn dev, pnpm dev, bun dev
```

Ouvrez [http://localhost:3000](http://localhost:3000) dans votre navigateur pour voir le résultat.

## Structure du projet

- `app/` : pages principales (Accueil, Projets, Contact)
- `components/` : composants réutilisables (Header, Footer, animations Framer Motion)
- `public/img/` : images du portfolio et du contact
- `data/projects.json` : données des projets affichés

## Déploiement

Le site peut être déployé facilement sur **Vercel** :

```bash
vercel
```

Pour plus d'informations, consultez la documentation de Next.js.
